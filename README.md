## Overview
This project analyzes retail sales data using SQL in PostgreSQL (pgAdmin 4).  
It focuses on data cleaning, exploratory analysis, and answering business-related questions to understand sales trends and customer behavior.

## Tech Stack
- PostgreSQL  
- pgAdmin 4  
- SQL  

## Dataset Schema
The dataset contains transaction-level retail sales data with the following columns:

- transactions_id  
- sale_date  
- sale_time  
- customer_id  
- gender  
- age  
- category  
- quantity  
- price_per_unit  
- cogs  
- total_sale  

## Analysis Summary
- Cleaned data by removing records with missing values  
- Analyzed sales performance by category  
- Identified high-value transactions and top customers  
- Studied monthly sales trends and order distribution by time of day  

## Usage
- Set up the database in PostgreSQL  
- Run the SQL scripts using pgAdmin 4  
- Modify queries to explore additional insights
